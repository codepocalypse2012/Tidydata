This Codebook is for week four of Getting and Cleaning Data through Coursera via Johns Hopkins taken from UCI.

The data produced by running the code in the repository is called sdata.

It contains 88 variables and 180 rows.

Subject: A character variable with 30 levels 1:30. Identifies the subject number that performed each action.
Activity: A character variable with 6 levels:"WALKING" "WALKING_UPSTAIRS"   "WALKING_DOWNSTAIRS" "SITTING" "STANDING" "LAYING"  
Variables 3 through 88: The mean of each measure performed by each subject. The mean and standard deviation were aggregated into a single average for each subject/activity pair.
